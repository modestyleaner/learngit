下面是简单的git笔记：

1 创建一个git库
   在一个合适的目录，最好没有中文字符：
     git init
	 该命令会生成一个默认隐藏的.git 文件夹，最好不要改动里面的东西
	 
2 添加文件到git仓库分2步：
   <1> git add filename
   <2> git commit -m "descriptinong about the change"
   
   
3 git 常用命令   
   git status 改命令用来查看当前的状态   
   git diff 可以查看修改了，但是并未提交的内容
   
  git log 查看历史更改记录  q 退出记录列表
